## 사용자 역할별 플로우

### 역할 1: 신규 고객
```
신규 고객 → 회원가입 → 이메일 인증 → 로그인 → 새 주문 생성 → 결제
    │
    └─ 주문 확정 → 샘플 업로드 → 샘플 결과 확인 → 본보정 진행 결정
       │
       └─ YES → 본보정 파일 업로드 → 결과 다운로드 → 완료
```

### 역할 2: 기존 고객 (재주문)
```
기존 고객 → 로그인 → 주문 목록 조회 → 주문 상태 확인
    │
    ├─ 진행중인 주문 있음 → 진행 상태 확인
    │
    └─ 새 주문 생성 → 결제 → 위와 동일한 플로우
```

### 역할 3: 전문가 (관리자)
```
전문가 대시보드 (미구현)
    │
    ├─ 샘플 업로드 대기 건 확인
    │  └─ 샘플 보정 작업 → 완료
    │
    ├─ 본보정 파일 확인
    │  └─ 본보정 작업 → 완료
    │
    └─ 실패 항목 관리
       └─ FailedItemManagementPage
```
